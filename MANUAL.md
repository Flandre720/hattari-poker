# 🪳 はったりポーカー オンライン — 遊び方ガイド

---

## 🎴 はったりポーカーとは？

はったりポーカーは、**ブラフ（はったり）** がカギとなるカードゲームです。
相手に嫌われ者の生き物カードを押し付け合い、**同じ種類のカードを4枚集めてしまったプレイヤー** が脱落します。
最後まで生き残ったプレイヤーが勝利！

> **対応人数**: 2〜6人  
> **プレイ時間**: 15〜30分  
> **必要なもの**: ブラウザのみ（アプリ不要）

## 🚀 ゲームの始め方

### exe版（配布版）の場合
1. `cockroach-poker.exe` をダブルクリック
2. ブラウザで表示されたURLを開く
3. `cloudflared.exe` が同梱されていれば、インターネット公開URLも自動で表示されます

### 開発版の場合
1. `npm run dev` でフロントエンド起動
2. 別ターミナルで `npx tsx server/server.ts` でサーバー起動

### ルーム作成・参加
1. **名前を入力** してルームを作成、または既存ルームのIDを入力して参加
2. ホスト（部屋を作った人）が **「ゲーム開始」** を押すとスタート
3. ルームIDを友達に共有すれば、同じ部屋に入れます（📋ボタンでコピー）

## 🎮 基本ルール

### ゲームの流れ

```
①カードを選ぶ → ②相手を選ぶ → ③宣言する → ④相手が判断する → ⑤結果
```

### 詳しい流れ

#### ① あなたのターン（カードを渡す側）
1. **手札からカードを1枚** タップして選ぶ
2. **渡す相手** を選ぶ（上のプレイヤーパネルをタップ）
3. **生き物の種類を宣言** する（本当のことを言っても、嘘をついてもOK！）
   - 例: 「ゴキブリ」のカードを持って「これは **ネズミ** だ」と嘘をつく

#### ② 相手のターン（カードを受けた側）
カードを渡された相手には **3つの選択肢** があります：

| 選択肢 | 説明 |
|--------|------|
| ❌ **嘘だと思う** | 宣言が嘘だと思ったらこれ |
| ✅ **本当だと思う** | 宣言が本当だと思ったらこれ |
| 👀 **確認して回す** | カードの正体を見てから、別のプレイヤーに回す |

#### ③ チャレンジの結果
- **予想が当たった** → 送り手がカードを引き取り、テーブルに表向きで置く
- **予想が外れた** → 受け手がカードを引き取り、テーブルに表向きで置く

#### ④ 「確認して回す」を選んだ場合
- カードの **正体を確認** できる
- その後、**まだカードを見ていない別のプレイヤー** を選んで回す
- このとき、**宣言をやり直す**（嘘をつき直してもOK）
- ※ 全員が既に見ていたら、チャレンジするしかありません

### 敗北条件

**同じ種類のカードがテーブルに4枚溜まったら脱落！**

例: テーブルに 🪳🪳🪳🪳 が4枚 → ゴキブリ4枚で脱落

### 勝利条件

**他の全員が脱落したら、最後の一人が勝利！** 🏆

## 🌀 ゲームモード

ルーム作成時に3つのモードから選べます：

### 🟢 ノーマルモード
基本ルールのみのシンプルなモード。初めての方におすすめ。

### 🟡 イベントモード
数ターンごとに **ランダムイベント** が発動！戦略が大きく変わります。

| 絵文字 | イベント名 | 効果 |
|:---:|----------|------|
| 🔀 | シャッフル | 全員の手札がシャッフルされて再配布される |
| 👁️ | リーク | ランダムなプレイヤーの手札が1枚公開される |
| 🛡️ | バリア | チャレンジに失敗してもカードを受け取らない |
| 🎲 | ルーレット | カードの送り先がランダムに決まる |
| ⚡ | ダブルリスク | チャレンジ失敗時、追加でランダム1枚テーブルに出る |
| 🔒 | ロック | 指定された1種類しか宣言できなくなる |
| 🙏 | 救済 | テーブルカードが多いプレイヤーが1枚を他者に送れる |

イベント間隔は **1〜5ターンごと** または **ランダム（20%確率）** から選べます。

### 🔴 スキルモード
**SP（スキルポイント）** を貯めて、4種の特殊スキルが使えます！

#### SPの貯め方
| 条件 | 獲得SP |
|------|--------|
| チャレンジ成功（嘘を見抜いた） | +2 |
| ブラフ成功（嘘を信じさせた） | +1 |
| 正直宣言でチャレンジを誘い成功 | +1 |

#### スキル一覧（最大SP: 5）

| スキル | コスト | 使えるタイミング | 効果 |
|--------|:---:|-----------------|------|
| ⚔️ **アタック** | 3 SP | カードを受けた時 | チャレンジ成功時、カードをテーブルカードが最も少ないプレイヤーに送る |
| 🔀 **チェンジ** | 2 SP | カードを受けた時 | カードの送り先を別のプレイヤーに変更する |
| 🛡️ **シールド** | 4 SP | カードを確認した後 | パス先のプレイヤーがチャレンジできなくなる |
| 💚 **ヒール** | 5 SP | 自分のターン開始時 | テーブルのカード1枚を手札に戻す |

## ⏱️ ターン制限時間

ルーム作成時に制限時間を設定できます（60秒〜300秒、60秒刻み）。
時間切れになると自動でアクションが実行されます。

---

## 💬 チャット機能

ゲーム中にチャットで会話できます。右下の💬ボタンからメッセージを送信。
ブラフの駆け引きに使うも良し、雑談に使うも良し！（50文字まで）

---

## 👁️ 観戦モード

ゲームに参加せず「**観戦者**」としてルームに入ることもできます。
観戦者は全プレイヤーの状況を見ることができますが、手は出せません。

---

## 📊 ゲーム終了時の称号

ゲーム終了後、プレイ内容に応じて **称号** が贈られます。
また、**リプレイログ** でゲームの全展開を振り返ることもできます。

## ⚙️ 設定

画面右上の **⚙️ 歯車ボタン** から各種設定が変更できます：

| 設定項目 | 説明 |
|---------|------|
| 🔊 BGM音量 | 背景音楽の音量調整 |
| 🔊 SE音量 | 効果音の音量調整 |
| 🔔 通知音 | ON/OFF切り替え |
| ⚡ アニメ速度 | 普通/速い |
| 🌙 背景暗さ | 0〜90で調整 |
| 🔤 フォントサイズ | 小/中/大 |
| 💬 チャット通知 | ON/OFF切り替え |

## 🌐 インターネット対戦について

### LAN内（同じWi-Fi）で遊ぶ場合
サーバー起動時に表示される `http://192.168.x.x:3001` のURLを共有してください。

### 遠くの友達と遊ぶ場合
`cloudflared.exe` が同梱されていれば、起動時に自動で**公開URL**が発行されます。
そのURLを共有するだけで、アカウント作成不要でどこからでも参加できます！

## 💡 攻略のコツ

1. **嘘と本当を混ぜる** — 常に嘘をつくと見破られやすい。時々本当のことを言おう
2. **テーブルカードを見る** — 相手のテーブルに既に3枚ある生き物は、押し付けるチャンス！
3. **「確認して回す」を活用** — カードの正体を知れば有利に嘘がつける
4. **相手のクセを読む** — ブラフの傾向を読み取ろう
5. **スキルモードでは SPを温存** — ヒール（5SP）は逆転の切り札！
