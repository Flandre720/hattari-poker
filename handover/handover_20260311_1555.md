# 引き継ぎドキュメント — 2026/03/11 15:55

## 概要
ゴキブリポーカー（Cockroach Poker）のPhase1 MVPを新規作成。ローカルマルチプレイ（ホットシート方式）でゲームの基本ロジックとUIを実装した。また、AI開発支援ツール（doc-readerスクリプト、setupワークフロー）も追加した。

## 変更内容

### 新規プロジェクト: cockroach-poker/
- **技術スタック**: Vite + React 19 + TypeScript + Vanilla CSS
- **ゲームロジック**: `src/core/types.ts` — 型定義、`src/core/gameEngine.ts` — Reducer方式のゲームエンジン
  - デッキ生成（8種×8枚=64枚）、Fisher-Yatesシャッフル、均等配布
  - 全アクション処理: カード選択→宣言→チャレンジ/パス
  - 敗北判定（同種4枚）、勝利判定（最後の1人）
- **UIコンポーネント**: SetupScreen, GameBoard, CardComponent, DeclarationModal, GameOverScreen
- **ホットシート方式**: プレイヤー交代時に端末引継ぎ画面を表示

### AI開発支援ツール追加
- `.agents/scripts/doc-reader.mjs` — .docx/.pdf のテキスト抽出スクリプト
- `.agents/scripts/package.json` — mammoth, pdf-parse の依存定義
- `.agents/workflows/setup.md` — 新規端末セットアップガイド（npm install手順も追記）

### 仕様書の読み取り
- `Oracles Protocol/docx/ゴキブリポーカー_オンライン対戦仕様書.docx` をdoc-readerで読み取り、`.md`も生成済み

## 技術的な注意点
- **verbatimModuleSyntax**: tsconfig.app.jsonで`verbatimModuleSyntax: true`が有効。型のimportは必ず`import type { ... }`で分離すること。これを怠るとViteのランタイムで型が見つからずサイレントに壊れる
- **起動方法**: `cockroach-poker/` ディレクトリで `npm run dev`
- **devサーバーポート**: デフォルト 5173（テスト時は5190を使用）
