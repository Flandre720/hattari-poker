## 技術的な注意点
- **verbatimModuleSyntax**: `tsconfig.app.json`で有効。型のimportは必ず `import type { ... }` で分離すること
- **サーバー起動**: `cockroach-poker/server/` で `npx tsx server.ts`（ポート3001）
- **クライアント起動**: `cockroach-poker/` で `npm run dev`（ポート5173）
- **Viteプロキシ**: `/socket.io` パスはViteがポート3001に自動転送
- **既存のホットシートコンポーネント**: `GameBoard.tsx`, `GameOverScreen.tsx`, `SetupScreen.tsx`, `gameEngine.ts`, `types.ts` はそのまま残置（オフラインモード復活時に再利用可能）
- **shared/types.ts とsrc/core/types.ts の関係**: 現在は `shared/types.ts` がオンラインで使用、`src/core/types.ts` はレガシーとして残置。統合を検討してよい
